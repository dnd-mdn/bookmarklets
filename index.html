    Getting started

    Select a script in the Explorer to preview its source code and prepare a bookmarklet.

    Using a bookmarklet

    • Drag the blue button into your browser's bookmarks bar.
    • Navigate to a page the script supports, then click the bookmark to run it.
    • If your browser hides the bookmarks bar, enable it first so you can drop the bookmarklet there.

    Tips

    • Some bookmarklets only work on specific websites.
    • You can inspect the code in the editor before adding the bookmarklet.
